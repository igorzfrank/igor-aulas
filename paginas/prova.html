⏰ Horário da Prova
A prova poderá ser realizada até o horário de saída dos alunos. Administre bem o seu tempo! Ao terminar, chame o professor para salvar seu projeto.
📋 PASSO A PASSO — Preparação e Execução
1 Instale o FIGMA no computador
Acesse o site oficial do Figma e faça o download da versão desktop. Instale o programa no seu computador antes de iniciar a prova.
2 Baixe seu repositório na máquina
Acesse o GitHub e faça o download (clone) do repositório da prova para o seu computador. Você precisará dos arquivos locais para trabalhar.
3 Faça o download dos Assets
Baixe o arquivo compactado com todos os recursos necessários para a prova (imagens, ícones e arquivos complementares).
📥 DOWNLOAD DOS ASSETS (WinRAR)
4 Faça o download do Caderno de Prova
Baixe o caderno de prova em PDF com todas as instruções detalhadas, requisitos e critérios de avaliação. Use-o como guia durante toda a prova.
📄 DOWNLOAD DO CADERNO DE PROVA
5 Acesse o link do FIGMA (COM INTERNET)
Este link deve ser acessado enquanto você tem acesso à internet. Depois de aberto no FIGMA desktop, o arquivo ficará disponível offline.
🔗 ABRIR FIGMA — CATÁLOGO HTML & CSS
🖼️ Resultado Esperado do Projeto
Abaixo está a imagem de referência do projeto que você deve entregar. Use-a como guia visual para conferir se seu resultado está de acordo com o esperado.
6 Ao terminar, chame o professor
Quando finalizar o projeto, chame o professor para salvar seu trabalho. Não feche o FIGMA nem o editor de código antes da conferência.
💡 DICAS IMPORTANTES
✅ Organize-se: Leia o caderno de prova completo antes de começar.
✅ FIGMA offline: Depois de abrir o link com internet, o FIGMA mantém o arquivo em cache.
✅ Salve sempre: Pressione Ctrl+S frequentemente para não perder seu progresso.
✅ Administre o tempo: A prova vai até o horário de saída — planeje cada etapa.
✅ Conferência final: Revise seu código antes de chamar o professor.
⚠️ Atenção: A prova é individual. O descumprimento das regras (uso de celular, internet ou comunicação com colegas) resultará em anulação da prova.
